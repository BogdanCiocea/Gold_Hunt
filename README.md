
  # Gold Hunt

 # Objective:
    * Collect as many gold as you can and progress to the next levels (you must score 500 points each level)
    * Avoid the mines (they can be shot down with the lasers)
    * Check for easter eggs

 # Controls:

    * Moving:
        W - up
        A - left
        S - down
        D - walking
    
    * Shooting:
        4 - left
        8 - up
        6 - right
        2 - down
    
    * Playlist:
        9 - forward
        7 - backward

 # Have fun!
